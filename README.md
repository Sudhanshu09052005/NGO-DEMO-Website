# NGO-DEMO

Simple one-page NGO website (SaharaRise Foundation demo) with sections: Home, About, Programs, Services, Gallery, Income & Impact, Donate, Contact.

How to publish:

- Add GitHub remote and push (replace USERNAME):

```powershell
git remote add origin https://github.com/USERNAME/NGO-DEMO.git
git push -u origin main
```

Or use `gh repo create NGO-DEMO --public --source=. --remote=origin --push` after installing and authenticating GitHub CLI.

Assets: local images are in the `images/` folder.
